# optimizing-USV-control
Codebase for "Optimizing unmanned surface vehicle control: A data-enabled learning approach"


## usvnuflcontroller

MATLAB Code Generated ROS node with surge speed and yaw rate controller.

To modify PID gains, access UsvNuFLController_data.cpp and change values of:
- Surge speed PID: Pu, Iu, Du
- Yaw rate PID: Pr, Ir, Dr
